王晨在深入学习贯彻习近平总书记关于坚持和完善人民代表大会制度的重要思想加强和改进人大代表工作交流会上强调

学深悟透融会贯通真信笃行

不断提升人大代表工作水平

新华社北京12月22日电 “深入学习贯彻习近平总书记关于坚持和完善人民代表大会制度的重要思想、加强和改进人大代表工作交流会”22日在京召开。中共中央政治局委员、全国人大常委会副委员长王晨出席会议并讲话。他强调，要把学习贯彻习近平新时代中国特色社会主义思想进一步引向深入，真正学深悟透、融会贯通、真信笃行，用实际行动贯彻落实党的十九届四中全会精神，推动各级人大统一思想认识，不断提升人大工作和代表工作水平。

王晨指出，深入学习贯彻习近平总书记关于坚持和完善人民代表大会制度的重要思想，是各级人大的重要政治任务，是做好新时代人大工作的根本保证。我们要深刻领会、准确把握其科学内涵，逐项贯彻落实到人大工作的全过程、各方面，自觉用以指引和推进新时代人大制度和人大工作不断取得新进展新成效。

王晨强调，要增强“四个意识”，坚定“四个自信”，做到“两个维护”，坚持党对人大工作的领导。进一步密切同人大代表的联系，尊重代表主体地位，健全代表联络机制，支持和保证代表依法履职，更好发挥人大代表作用。各级人大代表要提高政治站位，珍视代表职务，严格要求自己，依法履职尽责，自觉接受监督，当好党和国家密切联系人民群众的桥梁和纽带，做到忠诚为党分忧、忠实为民代言，不负党和人民的信任与重托。
